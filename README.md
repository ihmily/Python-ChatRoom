# Python-ChatRoom
基于Python socket UDP和Tkinter编写的多功能即时聊天室，功能包括有账号注册和登录，登录成功后可以查看在线用户，并和聊天室内的其他在线用户聊天，包含私聊和群发，能发送文字、表情包，以及文件等等

&emsp;

## 支持功能 

-  登录注册
-  显示在线用户
-  群聊和私聊
-  发送文字消息和表情包
-  发送文件
-  其他功能可自行添加
